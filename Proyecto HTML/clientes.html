  • Clientes

    Este campo muestra el nombre del cliente, que es un enlace a una página de detalles específicos para ese cliente.

  • Descripcion

    Aquí se pueden agregar una breve descripción o resumen sobre el cliente. Esto puede incluir información relevante sobre su empresa, industria o cualquier detalle importante que necesites recordar.

  • Fecha de adquisión

    Esta columna muestra la fecha en que adquiriste al cliente o comenzó tu relación con él. Esto puede ser útil para realizar un seguimiento de cuánto tiempo has trabajado con cada cliente.


Esta lista de clientes te permite tener una vista rápida de tus contactos y acceder a información detallada haciendo clic en el nombre de cada cliente. Puedes personalizar y añadir más detalles según las necesidades de tu negocio y tu CRM.

Para buscar más clientes siempre puedes visitar teamleader, una página web que explica de alguna manera cómo conseguir más clientes.
